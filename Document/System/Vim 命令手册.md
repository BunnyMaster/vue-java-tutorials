## 1. 基础移动（Navigation）

### 字符移动
- `h` ← / `j` ↓ / `k` ↑ / `l` →  
- 方向键也能用，但 Vim 提倡用 `hjkl` 保持手不离键盘

### 单词移动
- `w` → 跳到下一个单词开头  
- `W` → 跳到下一个单词开头（忽略符号）  
- `b` ← 跳到上一个单词开头  
- `B` ← 跳到上一个单词开头（忽略符号）  
- `e` → 跳到当前单词末尾  
- `E` → 跳到当前单词末尾（忽略符号）

### 行内跳转
- `0` 跳到行首  
- `^` 跳到第一个非空字符  
- `$` 跳到行尾  
- `f<char>` 向后搜索字符（如 `fa` 跳到下一个 `a`）

### 屏幕移动
- `Ctrl + f` 向下翻页  
- `Ctrl + b` 向上翻页  
- `Ctrl + u` 上翻半屏  
- `Ctrl + d` 下翻半屏  
- `gg` 跳到文件开头  
- `G` 跳到文件末尾  
- `H` 跳到屏幕顶部  
- `M` 跳到屏幕中间  
- `L` 跳到屏幕底部  
- `:<行号>` 跳转到指定行（如 `:42` 跳到第 42 行）

## 2. 编辑文本（Editing）

### 插入模式
- `i` 在光标前插入  
- `a` 在光标后插入  
- `I` 在行首插入  
- `A` 在行尾插入  
- `o` 在下一行插入  
- `O` 在上一行插入  
- `R` 进入替换模式（覆盖字符）  
- `Ctrl + h` 删除前一个字符（退格键）  
- `Ctrl + w` 删除前一个单词  
- `Ctrl + u` 删除到行首

### 删除操作
- `x` 删除当前字符  
- `X` 删除前一个字符  
- `dw` 删除一个单词  
- `db` 删除到上一个单词开头  
- `dd` 删除整行  
- `D` 删除从光标到行尾  
- `d$` 删除到行尾  
- `d^` 删除到行首  
- `dG` 删除到文件末尾  
- `dgg` 删除到文件开头

### 复制/粘贴
- `yy` 复制当前行  
- `yw` 复制一个单词  
- `y$` 复制到行尾  
- `yiw` 复制当前单词  
- `p` 粘贴到光标后  
- `P` 粘贴到光标前

### 撤销/重做/重复
- `u` 撤销  
- `Ctrl + r` 重做  
- `.` 重复上一次操作

### 其他编辑命令
- `J` 合并下一行到当前行  
- `>>` 向右缩进  
- `<<` 向左缩进  
- `==` 自动缩进当前行

## 3. 选择与可视化模式（Visual Mode）

- `v` 进入字符选择模式（按字符选择）  
- `V` 进入行选择模式（按行选择）  
- `Ctrl + v` 进入块选择模式（矩形选择）  
- `ggVG` 或 `ggvG` 全选整个文件  
- `o` 切换选择的光标端

### 选中后操作
- `d` 删除（剪切）选中内容  
- `y` 复制选中内容  
- `c` 修改（删除并进入插入模式）  
- `~` 切换选中内容的大小写  
- `>` 向右缩进选中内容  
- `<` 向左缩进选中内容

## 4. 搜索与替换（Search & Replace）

### 搜索
- `/pattern` 向前搜索  
- `?pattern` 向后搜索  
- `n` 下一个匹配项  
- `N` 上一个匹配项  
- `*` 向前搜索当前光标下的单词  
- `#` 向后搜索当前光标下的单词  
- `:noh` 临时取消搜索高亮

### 替换
- `:%s/old/new/g` 全局替换  
- `:%s/old/new/gc` 全局替换（每次确认）  
- `:s/old/new/g` 当前行替换

## 5. 组合命令（Power Moves）

- `di"` 删除引号内的内容  
- `ciw` 修改当前单词  
- `dt<char>` 删除直到某个字符（如 `dt)` 删除到 `)`）  
- `.` 重复上一次操作

## 6. 窗口管理

### 窗口分割
- `:sp [file]` 水平分割窗口  
- `:vsp [file]` 垂直分割窗口

### 窗口导航
- `Ctrl + w h` 移动到左侧窗口  
- `Ctrl + w j` 移动到下方窗口  
- `Ctrl + w k` 移动到上方窗口  
- `Ctrl + w l` 移动到右侧窗口  
- `Ctrl + w w` 循环切换窗口

### 窗口操作 
- `Ctrl + w c` 关闭当前窗口  
- `Ctrl + w o` 只保留当前窗口  
- `Ctrl + w +` 增加窗口高度  
- `Ctrl + w -` 减少窗口高度  
- `Ctrl + w >` 增加窗口宽度  
- `Ctrl + w <` 减少窗口宽度  
- `Ctrl + w =` 均衡窗口尺寸

## 7. 标签页管理

- `:tabnew [file]` 新建标签页  
- `gt` 下一个标签页  
- `gT` 上一个标签页  
- `:tabclose` 关闭当前标签页  
- `:tabonly` 关闭其他标签页  
- `:tabmove [n]` 移动标签页到第 n 位置

## 8. 文件操作

- `:e file` 打开文件  
- `:w` 保存文件  
- `:w file` 另存为  
- `:saveas file` 另存为  
- `:q` 退出  
- `:q!` 强制退出不保存  
- `:wq` 保存并退出  
- `:x` 保存并退出（仅当有修改时）  
- `:r file` 插入文件内容到当前位置  
- `:r !command` 插入命令输出到当前位置

## 9. 设置与配置

- `:set nu` 显示行号  
- `:set nonu` 隐藏行号  
- `:set hlsearch` 高亮搜索结果  
- `:set nohlsearch` 取消高亮搜索结果  
- `:set paste` 进入粘贴模式（防止自动缩进）  
- `:set nopaste` 退出粘贴模式

## 10. 其他实用命令

- `:!command` 执行外部命令  
- `:%!command` 用命令处理整个文件（如 `:%!sort` 排序）  
- `:help` 打开帮助文档  
- `Ctrl + g` 显示当前文件信息
