## 快速打开命令行

**Ctrl + Alt + T** - 最常用的快捷键，立即打开终端

## 命令行快捷

### 基础移动快捷键
- **Ctrl + A**：移动到**行首**（你已经知道）
- **Ctrl + E**：移动到**行尾**
- **Alt + F**：向前移动一个**单词**（在Mac上是 `Option + →`）
- **Alt + B**：向后移动一个**单词**（在Mac上是 `Option + ←`）
- **Ctrl + F**：向前移动一个**字符**（相当于 → 键）
- **Ctrl + B**：向后移动一个**字符**（相当于 ← 键）

### 编辑相关快捷键
- **Ctrl + U**：剪切从光标到**行首**的内容
- **Ctrl + K**：剪切从光标到**行尾**的内容
- **Ctrl + W**：剪切前一个**单词**
- **Ctrl + Y**：粘贴刚才剪切的内容（粘在光标位置）
- **Ctrl + D**：删除光标处的字符（相当于Delete键）
- **Ctrl + H**：删除光标前的字符（相当于Backspace键）
- **Ctrl + T**：交换光标前后的两个字符
- **Alt + T**：交换光标前后的两个单词
- **Ctrl + _** 或 **Ctrl + X Ctrl + U**：撤销上一次编辑

### 历史命令操作
- **Ctrl + P**：上一条命令（相当于 ↑ 键）
- **Ctrl + N**：下一条命令（相当于 ↓ 键）
- **Ctrl + R**：**反向搜索**历史命令（非常实用！）
  - 输入关键词搜索，按Ctrl+R继续搜索上一条匹配项
  - 按Enter执行，按→键将命令放入当前行编辑
- **Ctrl + G**：退出历史搜索模式（在Ctrl+R后使用）
- **Ctrl + S**：正向搜索历史命令（如果终端支持）
- **Ctrl + O**：执行当前找到的历史命令，并自动显示下一条

### 进程控制
- **Ctrl + C**：终止当前运行的程序
- **Ctrl + Z**：暂停当前程序（放入后台），可用 `fg` 恢复
- **Ctrl + L**：清屏（相当于 `clear` 命令）
- **Ctrl + S**：暂停屏幕输出（冻结终端）
- **Ctrl + Q**：恢复屏幕输出（解冻终端）

### 其他实用快捷键
- **Tab**：**自动补全**（按一次补全，按两次显示所有选项）
- **Alt + ?**：显示可能的补全列表（相当于按两次Tab）
- **Ctrl + XX**：在行首和当前光标位置间切换
- **Alt + U**：将当前单词转为大写
- **Alt + L**：将当前单词转为小写
- **Alt + C**：将当前单词首字母大写
