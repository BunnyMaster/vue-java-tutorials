# 📚 技术学习与实践项目集

这是一个用于个人学习和技能提升的代码仓库，包含各种技术栈的 demo 项目和练习代码。

## 🎯 项目用途

- **学习记录** - 记录技术学习过程中的实践代码
- **技能练习** - 通过实际项目巩固编程技能
- **代码参考** - 建立个人代码片段库，方便日后查阅
- **技术验证** - 测试新技术、框架和工具的功能特性

## 📁 项目目录说明

| 目录              | 技术栈          | 描述               |
| ----------------- | --------------- | ------------------ |
| `cloud-demo`      | 微服务          | 微服务相关演示     |
| `CSharp`          | C#              | C# 语言学习项目    |
| `Document`        | 文档            | 学习笔记和技术文档 |
| `drools`          | Drools          | 规则引擎学习       |
| `lang-chain-4j`   | LangChain4j     | Java AI 应用开发   |
| `mongo-db`        | MongoDB         | NoSQL 数据库实践   |
| `mq-demo`         | 消息队列        | 消息中间件演示     |
| `nacos`           | Nacos           | 服务发现和配置管理 |
| `quartz-demo`     | Quartz          | 任务调度框架       |
| `react`           | React           | 前端 React 框架    |
| `redis`           | Redis           | 内存数据库使用     |
| `spring-security` | Spring Security | 安全框架学习       |
| `vue2-tutorials`  | Vue2            | Vue2 教程和示例    |
| `vue3-tutorials`  | Vue3            | Vue3 教程和示例    |

## 💡 使用说明

1. 每个子目录都是独立的学习项目
2. 项目主要用于演示和学习目的
3. 代码可能包含实验性和未优化的内容
4. 欢迎参考和提出改进建议

## 🚀 技术栈覆盖

- **前端**: React, Vue2, Vue3
- **后端**: Java, C#
- **数据库**: MongoDB, Redis
- **中间件**: 消息队列, Nacos, Quartz
- **其他**: 规则引擎, AI 开发, 云服务
