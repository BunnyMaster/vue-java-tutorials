## 不同类型的使用场景对比

| 类型                   | 使用层                | 是否返回前端 | 包含业务逻辑 | 示例                  |
| :------------------- | :----------------- | :----- | :----- | :------------------ |
| **Entity**           | DAO/Repository     | ❌ 避免   | ❌ 否    | `UserEntity`        |
| **DTO**              | Controller/Service | ✅ 推荐   | ❌ 否    | `UserDTO`           |
| **VO**               | Controller         | ✅ 推荐   | ❌ 否    | `UserProfileVO`     |
| **BO**               | Service            | ❌ 避免   | ✅ 是    | `OrderBO`           |
| **Query**            | Controller/Service | ❌ 避免   | ❌ 否    | `UserQuery`         |
| **Form**             | Controller         | ❌ 避免   | ❌ 否    | `LoginForm`         |
| **Request/Response** | Controller         | ✅ 推荐   | ❌ 否    | `CreateUserRequest` |

| 场景 | 推荐返回类型 | 示例 | 说明 |
|------|------------|------|------|
| 获取单个资源 | `XXXDTO` | `UserDTO` | 最常用 |
| 获取列表 | `List<XXXDTO>` | `List<ProductDTO>` |  |
| 分页查询 | `PageDTO<XXXDTO>` | `PageDTO<OrderDTO>` |  |
| 复杂视图 | `XXXVO` | `DashboardVO` | 聚合数据 |
| 操作结果 | `XXXResult` | `LoginResult` | 特定操作 |
| 统一响应 | `ApiResponse<T>` | `ApiResponse<UserDTO>` | 包装响应 |
| 文件上传 | `UploadResult` | `UploadResult` | 文件相关 |
| 统计数据 | `XXXStatistics` | `SalesStatistics` | 统计类 |

## 核心原则总结

1. **对外接口（Controller）返回**：`DTO`、`VO`、`Response`、`Result`等
2. **对内传输（Service间）使用**：`DTO`、`BO`
3. **数据持久层**：`Entity`/`PO`
4. **参数接收**：`Request`、`Query`、`Form`、`Param`
5. **绝对禁止返回**：`Entity`、`BO`给前端

**简单记忆**：
- **向外吐**（返回给前端）：`DTO`、`VO`
- **向内吞**（接收参数）：`Request`、`Query`、`Form`
- **内部用**（Service层）：`BO`、`Entity`
