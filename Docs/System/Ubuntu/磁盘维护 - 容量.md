## 基础命令

### cd

```bash
# 进入用户主目录
cd

# 进入根目录
cd .

# 进入用户主目录
cd  ~

# 返回上级目录
cd ..

# 返回丙级目录
cd ,,/,,

# 把上个命令的参数作为cd参数使用
cd !$
```

### df

| 选项 | 说明                          |
| ---- | ----------------------------- |
| -a   | 包含全部文件系统              |
| -h   | 心可读性较高的方式显示        |
| -H   | 以1000Bytes为换算单位而非1024 |
| -i   | 显示inode信息                 |

**命令示例**

```bash
# 包含全部文件系统
df -a -h

# 查看根目录使用
df -a -h /
```

### du

| 选项          | 说明                   |
| ------------- | ---------------------- |
| -s            | 汇总显示，只显示总计   |
| -h            | 可读格式（KB, MB, GB） |
| -b            | 以字节为单位           |
| --max-depth=N | 显示到第 N 级子目录    |

**命令示例**

```bash
# 查看当前文件夹总大小（人类可读格式）
du -sh

# 查看精确字节数
du -sb

# 查看当前文件夹及所有子文件夹的大小
du -h --max-depth=1
```
