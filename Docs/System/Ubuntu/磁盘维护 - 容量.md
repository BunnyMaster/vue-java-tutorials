## 基础命令

### cd

```bash
# 进入用户主目录
cd

# 进入根目录
cd .

# 进入用户主目录
cd  ~

# 返回上级目录
cd ..

# 返回丙级目录
cd ,,/,,

# 把上个命令的参数作为cd参数使用
cd !$
```

### df

| 选项 | 说明                          |
| ---- | ----------------------------- |
| -a   | 包含全部文件系统              |
| -h   | 心可读性较高的方式显示        |
| -H   | 以1000Bytes为换算单位而非1024 |
| -i   | 显示inode信息                 |

```
# 包含全部文件系统
df -a -h

# 查看根目录使用
df -a -h /
```
